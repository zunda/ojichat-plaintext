# ojichat-plaintext
Your friendly ojisan in the cloud inspired by [ojichat-web](https://reverent-shirley-368990.netlify.com/).

## Running and accessing the app
### Local
```
$ go run main.go
```

Navigate to http://localhost:3000/

## As a Heroku app
```
$ heroku apps:create
$ git push heroku master
$ heroku apps:open
```
